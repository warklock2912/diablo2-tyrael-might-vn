# Belts

## Introduce

Belts can use Jeweller Belts to crafting more sockets. With Belts already used Jeweller Belts Max +1, that Belts still can use Jeweller to craft more socket Max +2 ( only roll socket , another opt not change)

## Craft Rate

| Socket Belt | Rate |
| ----------- | ---- |
| 0           | 45%  |
| 1           | 35%  |
| 2           | 20%  |
